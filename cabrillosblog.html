Parts of a Computer — What They Do

A quick, friendly guide to the main computer parts shown in the photo and their common uses.

Quick navigation

CPU (Processor)
Central Processing Unit
Acts as the brain of the computer. It performs calculations, processes data, and runs instructions that allow all programs to function.
CPU Cooler
Cooling System
Keeps the processor cool by dissipating heat. Prevents overheating and ensures stable performance.
Motherboard
Main Circuit Board
Connects all hardware components together and allows them to communicate — including the CPU, RAM, storage, and peripherals.
RAM
Random Access Memory
Provides temporary memory for running applications. The more RAM, the more tasks your computer can handle at once.
SSD
Solid State Drive
A fast storage device that greatly improves boot time and program loading speed compared to older hard drives.
HDD
Hard Disk Drive
Traditional storage device used to store large files like photos, videos, and documents at a lower cost per gigabyte.
PSU
Power Supply Unit
Supplies power to all computer parts. Converts electricity from the wall outlet into usable power for the motherboard and components.
Computer Case
System Enclosure
Houses and protects all computer components. It also provides airflow to keep them cool.
Expansion Cards
Add-on Components
Used to add extra functions like better graphics (GPU), sound, or network capabilities.
Monitor
Display Screen
Shows what the computer is doing. Displays images, text, and videos — an essential part of user interaction.
Keyboard
Input Device
Allows you to type letters, numbers, and commands to control the computer.
Mouse
Pointing Device
Used to move the pointer, click, and interact with elements on the screen.
Speakers
Audio Output
Produce sound from the computer — for music, videos, or system alerts.
